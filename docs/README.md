## 💡 设计理念

1. **混合模板方法**: 数据准确性（预计算）+ 语义理解（LLM）
2. **成本优化优先**: 使用 DeepSeek V3 降低成本，智能过滤输入数据
3. **健壮性**: 完善的错误处理、重试机制、部分结果支持
4. **国际化**: 原生支持中文输出

## 🎨 特色功能

- **智能 Issue 推荐**: 基于标签和活跃度过滤，推荐适合新手的贡献机会
- **置信度指标**: 每个分析部分都包含置信度评估（🟢高 🟡中 🔴低）
- **多级详细度**: 支持简短模式（快速概览）和详细模式（深入分析）
- **报告验证**: 自动检查报告完整性和格式正确性
